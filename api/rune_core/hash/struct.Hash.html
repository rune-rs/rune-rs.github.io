Struct Hash

Source 
#[repr(transparent)]
pub struct Hash(/* private fields */);
Expand description

The primitive hash that among other things is used to reference items, types, and native functions.

§Examples

use rune::Hash;

assert_ne!(Hash::index(0), Hash::index(1));
