Expand description

Capabilities specific to the textDocument/semanticTokens/* requests.

@since 3.16.0

Fieldsยง

ยงdynamic_registration: Option<bool>

Whether implementation supports dynamic registration. If this is set to true the client supports the new (TextDocumentRegistrationOptions & StaticRegistrationOptions) return value for the corresponding server capability as well.

ยงrequests: SemanticTokensClientCapabilitiesRequests

Which requests the client supports and might send to the server depending on the serverโs capability. Please note that clients might not show semantic tokens or degrade some of the user experience if a range or full request is advertised by the client but not provided by the server. If for example the client capability requests.full and request.range are both set to true but the server only provides a range provider the client might not render a minimap correctly or might even decide to not show any semantic tokens at all.

ยงtoken_types: Vec<SemanticTokenType>

The token types that the client supports.

ยงtoken_modifiers: Vec<SemanticTokenModifier>

The token modifiers that the client supports.

ยงformats: Vec<TokenFormat>

The token formats the clients supports.

ยงoverlapping_token_support: Option<bool>

Whether the client supports tokens that can overlap each other.

ยงmultiline_token_support: Option<bool>

Whether the client supports tokens that can span multiple lines.

ยงserver_cancel_support: Option<bool>

Whether the client allows the server to actively cancel a semantic token request, e.g. supports returning ErrorCodes.ServerCancelled. If a server does the client needs to retrigger the request.

@since 3.17.0

ยงaugments_syntax_tokens: Option<bool>

Whether the client uses semantic tokens to augment existing syntax tokens. If set to true client side created syntax tokens and semantic tokens are both used for colorization. If set to false the client only uses the returned semantic tokens for colorization.

If the value is undefined then the client behavior is not specified.
