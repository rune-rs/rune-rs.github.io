source ·
Expand description

This is a fork of the hashbrown crate.

It’s been modified to make use of the memory utilities provided by rune alloc.

This is a Rust port of Google’s high-performance SwissTable hash map, adapted to make it a drop-in replacement for Rust’s standard HashMap and HashSet types.

The original C++ version of SwissTable can be found here, and this CppCon talk gives an overview of how the algorithm works.
