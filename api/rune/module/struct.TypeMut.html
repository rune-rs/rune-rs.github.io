Struct rune::module::TypeMut

source · 
pub struct TypeMut<'a, T>
where
    T: ?Sized + TypeOf,
{ /* private fields */ }
Expand description

Handle to a a type inserted into a module which allows for mutation of its metadata.

This is returned by the following methods:

  • Module::ty - after a type has been inserted.
  • Module::type_meta - to modify type metadata for an already inserted type.
