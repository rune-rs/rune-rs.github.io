Module ::std::io

The std::io module contains a number of common things youâ€™ll need when doing input and output. The most core parts of this module are the [print()], [println()], and [dbg()] functions which are used to hook up printing for a Rune project.

With complete names:

  • ::std::io::print
  • ::std::io::println

Their definitions can be omitted from the built-in standard library, and can then easily be defined by third party modules allowing for printing to be hooked up to whatever system you want.
