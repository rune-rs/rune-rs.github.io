Function ::std::io::dbg(..)

Debug to output.

This is the actual output hook, and if you install rune modules without I/O enabled this will not be defined. It is then up to someone else to provide an implementation.

Examples

let number = 10;
let number = number * 4;

let who = "World";
let string = format!("Hello {}", who);

dbg(number, string);
