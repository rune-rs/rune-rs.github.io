Struct ::std::ops::Range

Type for a range expression.

A range expression is one of:

  • ..<value>
  • <value>..<value>
  • <value>..
  • ..=<value>
  • <value>..=<value>

Methods

fn contains<int>(self, value: int) -> bool

Test if the range contains the given integer.

fn iter(self) -> Iterator

Iterate over the range.

This panics if the range is not a well-defined range.
