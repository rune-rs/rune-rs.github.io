Struct std::fmt::Format

Overview

A format specification, wrapping an inner value.

Trait Implementations

impl std::fmt::Clone for Format
fn clone(value: any) -> any

Clone the specified value.

Examples

let a = 42;
let b = a;
let c = a.clone();

a += 1;

assert_eq!(a, 43);
assert_eq!(b, 42);
assert_eq!(c, 42);
